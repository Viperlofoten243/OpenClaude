## 🔑 Connect Your AI Provider

The application requires an API key to talk to external AI services. You obtain this key from the website of your chosen provider.

1. Create an account with a service like OpenRouter, OpenAI, or Gemini.
2. Find the section for API keys in your account settings.
3. Generate a new key.
4. Copy the key to your clipboard.
5. Paste this key into the settings menu inside OpenClaude.

The software saves these keys locally on your computer. It does not send your keys to any server other than the one you select.

## 🚀 How to Use the App

OpenClaude simplifies how you interact with AI. Once you connect your key, the app presents a list of available models.

1. Select your preferred provider from the dropdown menu.

You can switch between different models at any time. This allows you to compare answers from different providers like DeepSeek or Grok without changing apps.

## 🧩 FAQ

**Does this app store my data?**
The software stores settings and chat logs on your local hard drive. It does not store your data on a cloud server unless you choose a provider that does so.

**Can I use multiple providers at once?**
You can configure keys for multiple providers and switch between them in the settings as needed.

**What if the app fails to connect?**
Check your internet connection first. Then, verify that your API key is still active on the provider's website. If the key expires, the connection will fail.

**Is this official software?**
OpenClaude is a community tool. It connects to third-party services using their standard interfaces. It remains independent from the companies that own these AI models.

## 🛡 Security Practices

Keep your API keys confidential. Do not share your keys with others. If you suspect someone else has access to your key, revoke it immediately through your provider's dashboard and generate a new one. Update OpenClaude regularly to ensure you have the latest security patches and features. The application check for updates every time you launch it.

If you encounter issues, try these common fixes:

- Verify that your firewall allows OpenClaude to connect to the internet.
- Ensure you have the latest version of the app from the provided link.
